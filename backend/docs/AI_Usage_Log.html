Ketentuan Penggunaan AI:

Setiap kelompok wajib mencatat penggunaan AI secara transparan. AI boleh digunakan untuk brainstorming, desain database, debugging, dokumentasi, dan testing, tetapi hasilnya wajib diverifikasi dan dipahami oleh tim. Seluruh kode hasil AI telah di-review, dimodifikasi, dan diuji secara manual.

No Tanggal Nama Anggota Tools AI Prompt Penting Hasil dari AI Verifikasi / Revisi Tim
1 01 Jun 2026 Tim Kelompok Google Gemini Buatkan rancangan database untuk aplikasi monitoring kesehatan home care dengan entitas pasien dan monitoring kesehatan Saran struktur tabel: patients (patient_id, patient_name, nik_dummy, datebirth, gender, address, family_phone, patient_category) dan monitorings (id, patient_id, user_id, monitoring_date, blood_pressure, body_temperature, symptoms, notes, status) Direvisi: ditambahkan kolom respiratory_rate, oxygen_saturation, heart_rate sesuai kebutuhan klinis; nama kolom disesuaikan konvensi Laravel.
2 05 Jun 2026 Tim Kelompok Google Gemini Perbaiki validasi Laravel untuk input tekanan darah format "120/80" dan suhu tubuh rentang 35–42 derajat Celsius Contoh rules validasi: blood_pressure => ["required", "regex:/^\d{2,3}\/\d{2,3}$/"], body_temperature => ["required", "numeric", "min:35", "max:42"] Diuji manual dengan berbagai edge case (185/110, 34.5°C). Regex disesuaikan agar tidak terlalu ketat; rentang suhu dikonfirmasi sesuai standar klinis.
3 10 Jun 2026 Tim Kelompok Google Gemini Buat API endpoint Laravel GET /api/patients dan GET /api/patients/{id}/monitoring yang dapat diakses frontend PHP native Template controller API dengan method index() mengembalikan daftar pasien dengan relasi monitoring, method show() mengembalikan riwayat monitoring per pasien dalam format JSON Diuji via Postman dan browser. Endpoint publik (tanpa Sanctum) diverifikasi aman karena hanya read-only dan seluruh data bersifat dummy/simulasi.
4 15 Jun 2026 Tim Kelompok Google Gemini Buat README instalasi Laravel 12 dan panduan deploy ke InfinityFree shared hosting menggunakan FileZilla FTP Template README mencakup: requirements (PHP 8.3+, Composer, MySQL), langkah instalasi (clone, composer install, .env setup, migrate, serve), dan panduan struktur folder public_html Diedit sesuai aplikasi kelompok: ditambahkan langkah seed data dummy, catatan SESSION_DRIVER=file untuk InfinityFree, dan disclaimer data simulasi.
5 18 Jun 2026 Tim Kelompok Google Gemini Buatkan tampilan dashboard admin yang modern dengan sidebar dark navy, kartu statistik, dan tabel data pasien responsif menggunakan Bootstrap 5 dan custom CSS Template HTML/CSS: sidebar dark navy (#0F172A), stats cards dengan border-top warna berbeda per status, tabel dengan hover effect, topbar dengan pencarian global dan avatar pengguna Diimplementasikan ke Blade Laravel. Warna disesuaikan ke #001A42, ditambahkan filter tab status, validasi client-side JavaScript, integrasi data Eloquent ORM, dan Bootstrap Icons.
6 20 Jun 2026 Tim Kelompok Google Gemini Buat landing page frontend PHP native yang responsif sesuai desain Figma: navbar teal, hero section pencarian pasien, tentang layanan, cara kerja, dan footer ITSK Soepraoen Malang Template HTML+CSS landing page: navbar sticky glassmorphism, hero section putih-bersih dengan form pencarian kode pasien, seksi cara kerja 3 langkah berstep, dan footer informatif Diadaptasi ke PHP native dan dihubungkan ke API Laravel. Warna disesuaikan ke teal (#00B894), diuji responsif di mobile 375px dan desktop 1440px.
Catatan: Seluruh data pasien dan riwayat monitoring dalam aplikasi ini bersifat simulasi/dummy untuk keperluan akademik. Sistem ini tidak memberikan diagnosis medis. Rekomendasi yang dihasilkan hanya berupa tindak lanjut administratif.
